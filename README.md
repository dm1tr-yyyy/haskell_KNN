# k-NN Classifier/Regressor на Haskell

Реализация алгоритма k-ближайших соседей (k-NN) для задач классификации и регрессии с поддержкой различных метрик расстояния и выбора признаков.

## Возможности

- **Две задачи**: классификация и регрессия
- **Три метрики расстояния**: Euclidean, Manhattan, Chebyshev
- **Выбор признаков**: указание индексов столбцов для использования
- **Сохранение модели**: автоматическое сохранение обученной модели в файл
- **Подробный отчёт**: вывод предсказаний и статистики по качеству

## Требования

- Stack (рекомендуется) или GHC с cabal
- Haskell Platform

## Установка и сборка

```bash
# Клонирование репозитория
git clone <your-repo-url>
cd <project-directory>

# Сборка проекта
stack build

# Или с помощью cabal
cabal build

## Использование

stack run -- <обучение> <тест> <модель> [k] [метрика] [задача] [столбцы]
